# LessonFour

TaskFor Lesson12


Використовуючи форму hire me (з домашнього завдання до уроку №6) потрібно стилізувати елементи згідно дизайн системи: https://www.figma.com/community/file/874666139737585846

Додаткові вимоги:

    Потрібно модифікувати форму, а не створювати нову

    Форма має бути робоча (кнопка submit має відправляти форму, валідація на e-mail має працювати)

    Всі label-елементи мають бути привʼязані до відповідних полів форми

    Якщо елемент не валідний він має приймати відповідний стиль (див. дизайн)

    Завдання має бути оформлене в нову гілку, яка бере свій початок з main гілки. Після прийняття завдання відповідна гілка має бути влита в основну (main або master)

    Рамки в таблиці з контактними даними мають бути оформленні за допомогою CSS

    Основний блок документу повинен мати максимальну ширину 1024 px і повинен бути розташованим по центру

    Посилання з основного документу на форму має працювати у випадку, якщо він знаходиться в підпапці (наприклад, http://somesite.com/subfolder/your-cv/hire-me.html)
