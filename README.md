Використовуючи макет створіть свою UI систему:

    Опишіть кольори, які використовуються на сторінці та запишіть їх в змінні

    Замініть пряме використання кольорів у вашому коді, використовуючи новостворенні змінні

    Опишіть відступи, які використовуються на сторінці та винесіть їх в зміні також. Використайте новостворенні зміні у вашому коді

    Зробіть можливість використання тем для ваших документів. Для цього створіть окремий CSS-файл, в який винесіть ваші змінні. Огорніть зміні в клас (.main-theme) для того, щоб можна було їх приміняти лише до елементів з відповідним класом. Додайте клас main-theme до елементу body в вашому документі

    Створіть ще один CSS-файл, в якому використайте іншу кольорову палітру (можна використовувати цей ресурс для вибору кольорів, які комбінуються між собою). Нові зміни мають примінятись лише до елементів з класом my-theme

    Підключіть новий файл до документа. При додаванні класу main-theme до body має примінятись основна тема з макета, а при доданні класу my-theme має примінятись тема, яку ви створили

    Завдання оформіть в нову гілку вашого репозиторію та додайте посилання на HTML-документ у поле відповіді в особистому кабінеті
